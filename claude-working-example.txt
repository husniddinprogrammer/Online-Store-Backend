--------------------------------
1. ORDER PROFIT CALCULATION
--------------------------------

Requirement:
- Add totalProfit field to Order entity

Field:
- totalProfit (BigDecimal)

Calculation:

For each OrderItem:

discountedPrice = sellPrice * (1 - discount/100)
profit = (discountedPrice - arrivalPrice) * quantity

Total:
- Sum all item profits → save into Order.totalProfit

Rules:
- Use BigDecimal (NOT double)
- Handle discount correctly
- Avoid precision loss

--------------------------------
2. ORDER PAYMENT LOGIC
--------------------------------

Trigger:
- When user confirms order (checkout)

Behavior:

1. Get user balance
2. Calculate total order amount
3. If balance >= totalAmount:
    - Deduct balance
    - Set order status = PAID
    - Save order
4. If balance < totalAmount:
    - Throw exception: "Insufficient balance"

--------------------------------
3. DATA CONSISTENCY
